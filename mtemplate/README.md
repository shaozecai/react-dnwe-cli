# react-dnwe-cli
> React 移动端初始化工程

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm start
```

## Tools
- [React](https://react.docschina.org/) - 用于构建用户界面的 JavaScript 库。
- [Material-UI](https://material-ui.com/zh/) - React 组件用于更快速、更简便的 web 开发。
